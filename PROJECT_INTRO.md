# 微纳米气泡课题组智能Agent系统 — 项目介绍

## 一、项目概述

**"小气"** 是微纳米气泡课题组的 AI 智能助手，服务于约 20 人的研究实验室。

系统以 Claude API 为核心大脑，通过自然语言对话的方式，为课题组提供任务管理、会议组织、知识检索、成员协作等全方位智能化服务。支持网页端和微信端双重接入，课题组成员无论在电脑前还是手机上，都能便捷地与小气交互。

---

## 二、系统架构

### 技术栈

| 层级 | 技术选型 |
|------|---------|
| 后端框架 | Python 3.11 + FastAPI + SQLAlchemy (异步) |
| 数据库 | PostgreSQL + pgvector (向量搜索) |
| 前端 | Vue 3 + Element Plus + Vite + Pinia |
| AI 能力 | Claude API (mimo-v2.5 多模态) |
| 语音识别 | faster-whisper (GPU large-v3) |
| 语音合成 | Edge-TTS |
| 缓存与会话 | Redis |
| 文件存储 | MinIO (兼容 S3) |
| 任务队列 | Celery + Redis |
| 部署方式 | Docker Compose + FRP 内网穿透 |

### 部署拓扑

```
用户 → 云服务器 (Nginx + SSL) → FRP 隧道 → 本地电脑 (Docker 全服务 + GPU)
```

- **云服务器**（2核2G）：仅运行 Nginx 反向代理 + FRP 服务端 + webhook 接收服务，轻量无压力
- **本地电脑**：运行全部应用服务（app、PostgreSQL、Redis、MinIO、Whisper GPU、Celery）
- **FRP 隧道**：本地 8000 端口穿透到云服务器，用户通过 `https://agent.mnb-lab.cn` 访问
- **Webhook 自动部署**：GitHub push 到 main 分支 → 云服务器 webhook → SSH 到本地 Windows → 自动 git pull + docker compose restart app

## 三、核心功能模块

### 1. 智能对话 Agent

Agent 是整个系统的核心，拥有 **15 个工具能力**：

| 工具 | 能力 |
|------|------|
| `create_task` / `query_tasks` / `update_task` | 任务创建、查询、状态更新 |
| `create_meeting` / `query_meetings` | 会议创建、查询 |
| `query_members` | 成员信息查询 |
| `query_projects` / `generate_project_plan` | 项目管理、计划生成 |
| `search_knowledge` / `save_conversation_knowledge` | 知识库检索与保存 |
| `web_search` | 联网搜索（搜狗+必应双引擎） |
| `save_memory` / `search_memory` / `forget_memory` | 长期记忆存取 |
| `summarize_meeting_transcript` | 会议转录自动总结 |
| `query_all_member_tasks` | 全员任务状况查询（管理员/组长权限） |

**特色交互模式 — 先简要后详细**：
用户提问时，小气先立即返回【简要】核心结论，后台并行生成【详细】展开内容并自动追加到对话。兼顾了回答速度和内容完整性。

### 2. 任务管理

- **创建与分配**：管理员可将任务分配给任何成员，普通成员可管理自己创建或被分配的任务
- **自定义提醒**：支持多个自定义时间点，精确到分钟级别
- **主动提醒**：Celery 每 10 秒扫描 Redis 调度提醒，去重 24 小时内不重复通知
- **软删除/垃圾桶**：删除的任务进入垃圾桶，3 天后自动清除，支持恢复
- **视图优化**：任务列表优先显示"进行中"，同状态内按创建日期排序，负责人处显示头像

### 3. 企业微信集成

- **群机器人**：私聊或群聊 @机器人 均可触发，支持任务派发、进度回复、到期提醒
- **微信插件**：成员使用普通微信通过插件与小气对话，无需下载企业微信
- **双向通知**：任务创建和即将到期时，同时通知管理员（创建人）和负责人（ assignee）
- **身份识别**：支持 UserId / 昵称 / 手机号 / 外部用户（wx 开头）多信号识别

### 4. 知识库

- **语义搜索**：基于 pgvector + text2vec-base-chinese 的真实向量相似度搜索
- **文件上传**：支持 PDF / Word / Excel / TXT / Markdown，自动提取文本
- **AI 自动处理**：上传后自动生成摘要、关键词、分类标签
- **对话知识入库**：Agent 自动将对话中有价值的内容保存到知识库

### 5. 长期记忆

- **用户偏好记忆**：记住成员常用设置和习惯
- **对话摘要**：对话结束后后台自动提取值得记忆的信息
- **知识图谱**：从对话中提取人员-项目-成果关系
- **记忆检索**：新对话自动注入相关记忆到系统提示词

### 6. 语音交互

- **语音输入**：faster-whisper GPU 实时转文字，支持 SILK（微信语音）格式转换
- **语音播报**：Edge-TTS 将 Agent 回复转为语音
- **领域术语优化**：注入微纳米气泡专业术语，提升识别准确率
- **实时转写**：会议中实时语音转写为文字（WebSocket）

## 四、"小气"助手界面预览

### 对话窗口功能

```
┌─────────────────────────────────────────┐
│  🤖 小气助手                    [🎤] [🗑] │
├─────────────────────────────────────────┤
│                                         │
│  👤 你: 查看我的任务                    │
│                                         │
│  🤖 小气:                               │
│  【简要】                               │
│  您有 3 项进行中的任务：                │
│  1. zeta电位文献整理 (负责人: 张三)     │
│  2. 组会PPT制作 (负责人: 你)           │
│  3. 设备维护记录 (负责人: 你)           │
│                                         │
│  [点击查看详情 ▼]                       │
│                                         │
│  👤 你: [图片文件]                      │
│                                         │
│  🤖 小气: 已识别图片内容，正在分析...   │
│                                         │
├─────────────────────────────────────────┤
│  [📎] [📎] [输入消息...]          [发送]│
└─────────────────────────────────────────┘
```

### 支持的输入方式

- **文字输入**：Enter 发送，Shift+Enter 换行
- **语音输入**：点击 🎤 切换语音模式
- **图片上传**：点击 📎 选择图片，支持拖拽上传
- **文件上传**：支持 PDF/Word/Excel/TXT，自动解析内容后发送
- **快捷入口**：首页提供"我的任务"、"最近会议"、"项目进度"、"知识问答"快捷按钮

### 支持的回复形式

- **文字回复**：支持 Markdown 格式
- **简要+详细双层**：先显示核心结论，可展开查看完整分析
- **图片描述**：上传图片后自动识别并描述内容
- **文件解读**：上传文件后自动提取并分析内容
- **结构化输出**：任务列表、会议摘要、全员工作状况等结构化展示

## 六、安全与可靠性

- **JWT 认证**：全部 31 个 API 端点均需认证（除登录外）
- **登录限流**：滑动窗口限流器，5 分钟内最多 5 次尝试
- **CORS 收紧**：仅允许指定域名跨域访问
- **Docker 资源限制**：各服务设置了内存上限
- **Nginx 限流**：API 10r/s，登录 5r/m
- **结构化日志**：生产环境 JSON 格式日志，便于接入 ELK/Loki
- **数据库备份**：每日 pg_dump 备份，保留 7 天
