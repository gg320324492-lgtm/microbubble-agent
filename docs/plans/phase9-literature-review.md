# Phase 9: 自动化文献综述 — 详细实施计划

> 预计周期: **6-8 周** | 优先级: 🟡 中

## 9.1 文献自动获取

### 9.1.1 知网/万方 API 对接
| 任务 | 技术方案 | 工时 |
|------|----------|------|
| 知网 API 申请 | 联系知网申请学术接口 | 2d |
| 万方 API 对接 | 万方数据 API 集成 | 3d |
| 文献元数据提取 | 标题/作者/摘要/关键词 | 2d |
| 全文获取 | PDF 下载（需机构权限） | 2d |
| 去重处理 | DOI/标题去重 | 1d |

**验收标准**: 输入关键词自动检索知网/万方文献

### 9.1.2 PubMed/Google Scholar
| 任务 | 技术方案 | 工时 |
|------|----------|------|
| PubMed API | NCBI E-utilities API | 2d |
| Google Scholar | Scholarly 库爬取 | 3d |
| 文献筛选 | 按年份/影响因子/引用数 | 2d |
| 翻译处理 | 英文摘要自动翻译 | 1d |

**验收标准**: 支持英文文献检索和摘要翻译

### 9.1.3 RSS 订阅监控
| 任务 | 技术方案 | 工时 |
|------|----------|------|
| RSS 解析 | feedparser 解析期刊 RSS | 2d |
| 订阅管理 | 添加/删除/分类订阅源 | 2d |
| 新文献通知 | 推送新文献到通知系统 | 1d |
| 自动入库 | 新文献自动添加到知识库 | 2d |

**验收标准**: 订阅期刊后自动推送新发表文献

### 9.1.4 文献批量导入
| 任务 | 技术方案 | 工时 |
|------|----------|------|
| EndNote 导入 | 支持 .enw/.ris 格式 | 2d |
| Zotero 导入 | Zotero API 集成 | 2d |

| PDF 批量上传 | 上传 PDF 自动提取元数据 | 2d |

**验收标准**: 支持从 EndNote/Zotero 批量导入文献

## 技术风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 知网 API 申请困难 | 无法获取中文文献 | 兜底方案：手动上传 PDF |
| 文献版权问题 | 全文获取受限 | 仅获取摘要+元数据 |
| 综述质量不高 | 生成内容不够专业 | 人工审核+迭代优化 |
| 方法对比不准确 | 对比结果有误 | 人工确认+置信度标注 |
