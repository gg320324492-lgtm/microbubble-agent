# GPU 功能 01: 实验数据智能分析 — 详细实施计划

> 预计周期: **4-6 周** | 优先级: 🔴 高 | GPU 需求: ~4GB

## 功能概述

上传 Excel/CSV 实验数据后，AI 自动识别数据类型、生成统计图表、发现异常值、预测趋势、推荐最优参数。

## 数据库设计

```sql
-- 实验数据表
CREATE TABLE experiment_data (
    id SERIAL PRIMARY KEY,
    experiment_id INTEGER REFERENCES experiments(id),
    name VARCHAR(200) NOT NULL,
    description TEXT,
    file_path VARCHAR(500),
    file_type VARCHAR(20),
    columns JSONB,  -- 列定义 [{name, type, unit}]
    rows JSONB,     -- 数据行
    row_count INTEGER,
    version INTEGER DEFAULT 1,
    created_by INTEGER REFERENCES members(id),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- 分析结果表
CREATE TABLE analysis_results (
    id SERIAL PRIMARY KEY,
    data_id INTEGER REFERENCES experiment_data(id),
    analysis_type VARCHAR(50),  -- descriptive/inferential/regression/anomaly/prediction
    result JSONB,
    charts JSONB,  -- 图表配置
    report_path VARCHAR(500),
    created_at TIMESTAMP DEFAULT NOW()
);

-- 图表配置表
CREATE TABLE chart_configs (
    id SERIAL PRIMARY KEY,
    data_id INTEGER REFERENCES experiment_data(id),
    chart_type VARCHAR(50),
    config JSONB,
    preview_path VARCHAR(500),
    created_at TIMESTAMP DEFAULT NOW()
);
```

## API 设计

```python
# 数据上传
POST /api/v1/experiment-data/upload
# 请求: multipart/form-data {file, name, experiment_id}
# 响应: {id, name, columns, preview_rows}

# 数据预览
GET /api/v1/experiment-data/{id}/preview
# 响应: {columns, rows, statistics}

# 自动分析
POST /api/v1/experiment-data/{id}/analyze
# 请求: {analysis_type: "descriptive|inferential|regression|anomaly|prediction"}
# 响应: {analysis_id, result, charts}

# 图表生成
POST /api/v1/experiment-data/{id}/charts
# 请求: {chart_type, config}
# 响应: {chart_id, config, preview_url}

# 趋势预测
POST /api/v1/experiment-data/{id}/predict
# 请求: {target_column, periods, method}
# 响应: {predictions, confidence_intervals, chart}

# 参数优化
POST /api/v1/experiment-data/{id}/optimize
# 请求: {objectives, constraints, method}
# 响应: {optimal_params, pareto_front}

# 报告生成
POST /api/v1/experiment-data/{id}/report
# 请求: {template_id, format}
# 响应: {report_id, download_url}
```
