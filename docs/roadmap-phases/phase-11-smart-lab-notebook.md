# Phase 11: 智能实验记录本

> 周期：4-6 周 | 优先级：🔴 高

## 目标

构建结构化的实验记录系统，支持模板化记录、版本控制、智能搜索。

## 核心功能

### 11.1 结构化实验记录
- 实验目的、材料、步骤、结果、结论
- 支持富文本编辑（Markdown）
- 图片/文件附件
- 实验时间线

### 11.2 实验模板
- 预置常用实验模板
- 自定义模板创建
- 模板分享

### 11.3 版本控制
- 实验记录修改历史
- 版本对比（diff）
- 回滚到历史版本

### 11.4 智能搜索
- 全文搜索
- 按实验类型/日期/人员筛选
- 语义搜索（相似实验）

## 技术方案

### 前端
- Markdown 编辑器（已有 marked.js）
- 图片粘贴上传
- 版本历史时间线

### 后端
- 实验记录 CRUD
- 版本快照（每次保存创建快照）
- 全文搜索（PostgreSQL tsvector）

### 数据库
- `lab_notebooks` 表
- `lab_records` 表
- `lab_record_versions` 表

## 实现步骤

1. **Week 1-2**: 数据模型 + CRUD API
2. **Week 3-4**: 前端编辑器 + 模板
3. **Week 5**: 版本控制 + 对比
4. **Week 6**: 搜索 + 筛选

## 验收标准

- [ ] 支持 Markdown 编辑
- [ ] 图片可粘贴上传
- [ ] 版本历史可查看
- [ ] 版本对比功能正常
- [ ] 搜索响应 < 1 秒
