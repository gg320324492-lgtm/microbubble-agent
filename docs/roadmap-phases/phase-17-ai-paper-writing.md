# Phase 17: AI 辅助论文写作

> 周期：6-8 周 | 优先级：🟡 中

## 目标

提供论文写作的全方位 AI 辅助，包括大纲生成、内容建议、格式检查。

## 核心功能

### 17.1 大纲生成
- 根据研究主题生成论文大纲
- 章节结构建议
- 字数分配建议

### 17.2 内容建议
- 段落扩写建议
- 论证逻辑优化
- 学术用语推荐
- 引用插入建议

### 17.3 格式检查
- 学术规范检查（引用格式、图表编号）
- 语言风格检查（学术 vs 口语）
- 查重辅助（相似度检测）

### 17.4 写作模板
- 期刊论文模板
- 学位论文模板
- 会议论文模板
- 自定义模板

## 技术方案

### 内容生成
- Claude API（长文本生成）
- 提示词模板（分章节优化）

### 格式检查
- 规则引擎（格式规范）
- 语言模型（风格检查）

### 查重
- 文本相似度（余弦相似度）
- 引用检测

## 实现步骤

1. **Week 1-2**: 大纲生成 + 模板
2. **Week 3-4**: 内容建议 + 扩写
3. **Week 5-6**: 格式检查 + 查重
4. **Week 7-8**: 集成优化 + 导出

## 验收标准

- [ ] 大纲结构合理
- [ ] 内容建议可用
- [ ] 格式检查准确
- [ ] 查重结果可信
- [ ] 支持多种模板
