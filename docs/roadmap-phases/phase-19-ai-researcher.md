# Phase 19: 课题组专属 AI 研究员

> 周期：8-12 周 | 优先级：🟢 低

## 目标

构建能够自主进行科研工作的 AI 研究员，具备假设生成、实验设计、论文草稿能力。

## 核心功能

### 19.1 自主研究
- 研究问题识别
- 文献调研
- 假设生成
- 实验设计

### 19.2 假设验证
- 假设合理性评估
- 验证方案设计
- 结果分析
- 假设修正

### 19.3 论文草稿
- 自动生成论文草稿
- 图表生成
- 参考文献管理
- 格式化

### 19.4 知识积累
- 研究经验总结
- 最佳实践记录
- 失败案例分析

## 技术方案

### 研究流程
- LangChain Agent 框架
- 多步推理（Chain-of-Thought）
- 工具调用（搜索、计算、绘图）

### 知识管理
- 向量数据库（已有）
- 知识图谱（已有）
- 经验库（新增）

### 论文生成
- 长文本生成（Claude API）
- 模板填充
- 引用管理

## 实现步骤

1. **Week 1-3**: 研究流程框架
2. **Week 4-6**: 假设生成 + 验证
3. **Week 7-9**: 论文草稿生成
4. **Week 10-12**: 知识积累 + 优化

## 验收标准

- [ ] 能自主提出研究问题
- [ ] 假设生成合理
- [ ] 实验设计可行
- [ ] 论文草稿质量可用
- [ ] 知识积累有效
