# Phase 7: 多模态知识库

> 周期：4-6 周 | 优先级：🔴 高

## 目标

支持图片、公式、表格等非文本内容的识别和入库，扩展知识库的模态覆盖范围。

## 核心功能

### 7.1 图片识别入库
- 用户上传图片（实验照片、图表、截图等）
- OCR 提取图片中的文字内容
- LLM 分析图片内容，生成描述和标签
- 自动关联相关知识条目

### 7.2 公式识别与计算
- 支持 LaTeX/PNG 公式图片上传
- 公式 OCR 识别（Mathpix API 或本地模型）
- 公式结构化解析（变量、常量、运算符）
- 与现有公式库集成，支持安全计算

### 7.3 表格数据提取
- 表格图片识别（行列结构）
- 自动转换为结构化数据
- 支持 CSV/Excel 导出
- 数据可视化（图表生成）

## 技术方案

### 前端
- 图片上传组件（支持拖拽、粘贴、批量）
- 图片预览 + 识别结果展示
- 公式编辑器（LaTeX 实时预览）
- 表格编辑器（识别结果可手动修正）

### 后端
- OCR 服务：PaddleOCR / Tesseract
- 公式识别：Mathpix API / Pix2Tex
- 表格识别：TableTransformer
- 图片分析：Claude Vision API（已接入）

### 数据库
- `knowledge_items` 表新增字段：
  - `content_type`: text/image/formula/table
  - `media_url`: 媒体文件 URL
  - `ocr_text`: OCR 提取的文本
  - `structured_data`: 结构化数据（JSON）

## 实现步骤

1. **Week 1-2**: 图片上传 + Claude Vision 分析
2. **Week 3**: OCR 集成 + 文字提取
3. **Week 4**: 公式识别 + 公式库集成
4. **Week 5-6**: 表格识别 + 数据导出

## 验收标准

- [ ] 支持上传 JPG/PNG/WebP 图片
- [ ] 图片 OCR 准确率 > 90%
- [ ] 公式识别准确率 > 85%
- [ ] 表格识别准确率 > 80%
- [ ] 识别结果可手动修正
- [ ] 与现有知识库无缝集成
