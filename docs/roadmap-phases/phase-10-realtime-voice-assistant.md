# Phase 10: 实时语音科研助手

> 周期：6-8 周 | 优先级：🟡 中

## 目标

实现实时语音对话能力，支持语音问答、实时转录、多语言交互。

## 核心功能

### 10.1 实时语音对话
- 流式语音识别（WebSocket）
- 实时语音合成（Edge-TTS / VITS）
- 端到端延迟 < 2 秒

### 10.2 语音问答
- 语音提问 → 转录 → AI 回答 → 语音播报
- 支持打断（barge-in）
- 上下文保持（多轮对话）

### 10.3 多语言支持
- 中文/英文语音识别
- 自动语言检测
- 翻译功能（中↔英）

### 10.4 会议辅助
- 实时字幕显示
- 发言人识别
- 会议摘要自动生成

## 技术方案

### 语音识别
- 流式 ASR：Whisper Streaming / FunASR
- VAD：silero-vad（已有）
- 声纹：3D-Speaker（已有）

### 语音合成
- Edge-TTS（已有）
- VITS 本地模型（可选）

### 通信
- WebSocket 流式传输
- 音频格式：PCM 16kHz 16bit

## 实现步骤

1. **Week 1-2**: 流式 ASR 集成
2. **Week 3-4**: 语音合成 + 对话管理
3. **Week 5-6**: 打断功能 + 多轮对话
4. **Week 7-8**: 多语言 + 会议辅助

## 验收标准

- [ ] 端到端延迟 < 2 秒
- [ ] 识别准确率 > 95%
- [ ] 支持打断功能
- [ ] 多语言识别正常
- [ ] 会议实时字幕可用
