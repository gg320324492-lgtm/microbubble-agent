# Phase 13: 自动化文献综述

> 周期：6-8 周 | 优先级：🟡 中

## 目标

实现文献的自动检索、摘要生成、引用管理，辅助文献综述写作。

## 核心功能

### 13.1 文献检索
- 集成学术搜索引擎（Google Scholar、CNKI、万方）
- 关键词/语义检索
- 相关文献推荐

### 13.2 文献解析
- PDF 全文解析
- 提取标题、作者、摘要、关键词
- 识别引用关系

### 13.3 摘要生成
- AI 生成文献摘要
- 多文献对比分析
- 研究趋势识别

### 13.4 引用管理
- 自动生成参考文献格式（GB/T 7714、APA、IEEE）
- 引用插入（Word/LaTeX）
- 文献库管理

## 技术方案

### 文献检索
- SerpAPI（Google Scholar）
- CNKI 爬虫（需授权）

### PDF 解析
- PyPDF2 / pdfplumber
- GROBID（学术 PDF 解析）

### 摘要生成
- Claude API（长文本摘要）
- 提示词优化（学术风格）

## 实现步骤

1. **Week 1-2**: 文献检索 API 集成
2. **Week 3-4**: PDF 解析 + 元数据提取
3. **Week 5-6**: 摘要生成 + 对比分析
4. **Week 7-8**: 引用管理 + 导出

## 验收标准

- [ ] 支持中英文文献检索
- [ ] PDF 解析准确率 > 90%
- [ ] 摘要质量可用
- [ ] 引用格式正确
- [ ] 支持批量导入
