## Phase 7: 多模态知识库

**目标**: 从纯文本扩展到图文公式全模态理解

### 7.1 图片理解与提取
- [ ] PDF 图片提取并关联到知识条目（已有 `_upload_knowledge_images`）
- [ ] 图片 OCR 文字提取（PaddleOCR / Tesseract）
- [ ] 图片内容描述生成（Claude Vision API）
- [ ] 图片内嵌到知识详情页 Markdown 渲染

### 7.2 公式识别与结构化
- [ ] LaTeX 公式识别（Mathpix API / Pix2Tex）
- [ ] 公式自动提取入库（已有 `formula_service`）
- [ ] 公式编辑器前端组件
- [ ] 公式与知识条目双向关联

### 7.3 表格智能解析
- [ ] PDF 表格提取（Camelot / pdfplumber）
- [ ] 表格结构化存储（JSONB 格式）
- [ ] 表格可视化渲染（前端组件）
- [ ] 表格数据对比分析

### 7.4 多模态搜索
- [ ] 图片相似度搜索（CLIP embedding）
- [ ] 公式语义搜索（LaTeX → 向量）
- [ ] 跨模态关联（文字↔图片↔公式）

**预期产出**: 知识库支持上传图片/PDF/Word，AI 自动提取文字、图片、公式、表格并结构化存储

## 远期愿景: 闭环科研辅助

```
┌─────────────────────────────────────────────────────────────┐
│                    闭环科研辅助流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌──────────┐    ┌──────────┐    ┌──────────┐            │
│   │  假设生成  │───▶│  实验设计  │───▶│  数据分析  │            │
│   └──────────┘    └──────────┘    └──────────┘            │
│        ▲                                    │              │
│        │                                    ▼              │
│   ┌──────────┐    ┌──────────┐    ┌──────────┐            │
│   │  知识更新  │◀───│  结论提炼  │◀───│  结果验证  │            │
│   └──────────┘    └──────────┘    └──────────┘            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 闭环流程
1. **假设生成**: 基于知识图谱和文献空白，AI 自动生成可验证假设
2. **实验设计**: 基于假设和文献，AI 生成详细实验方案
3. **数据分析**: 实验数据上传后，AI 自动分析并验证假设
4. **结论提炼**: AI 提取关键结论，更新知识库
5. **知识更新**: 新知识融入图谱，触发新假设生成
6. **循环迭代**: 持续积累，形成课题组专属知识体系

## 技术架构演进

### 当前架构
```
用户 ──▶ Vue3 前端 ──▶ FastAPI ──▶ Claude API
                    ├──▶ PostgreSQL + pgvector
                    ├──▶ Redis
                    └──▶ Celery + Whisper
```

### 目标架构
```
用户 ──▶ Vue3 前端 ──▶ FastAPI ──▶ Claude API (主模型)
      │                  │     ├──▶ 本地微调模型 (领域专用)
      │                  │     └──▶ 多模态模型 (图片/公式)
      │                  ├──▶ PostgreSQL + pgvector
      │                  ├──▶ Redis + RabbitMQ
      │                  ├──▶ Celery + Whisper + OCR
      │                  └──▶ MinIO (文件存储)
      └──▶ 语音助手 ──▶ 实时 ASR/TTS
```

### 关键技术选型
| 模块 | 当前 | 目标 |
|------|------|------|
| LLM | Claude API | Claude + 本地微调模型 |
| Embedding | text2vec-base-chinese | 多模态 CLIP |
| ASR | faster-whisper | faster-whisper + 实时流式 |
| OCR | 无 | PaddleOCR |
| 公式识别 | 无 | Pix2Tex / Mathpix |
| 本地模型 | 无 | Qwen-7B / ChatGLM-6B |
