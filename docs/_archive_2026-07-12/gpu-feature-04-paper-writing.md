# GPU 功能 04: 科研写作助手 — 详细实施计划

> 预计周期: **4-6 周** | 优先级: 🟡 中 | GPU 需求: ~14GB

## 功能概述

AI 辅助论文写作、实验报告生成、基金申请书撰写，支持大纲生成、段落润色、引用建议、格式检查。

## 数据库设计

```sql
-- 论文草稿表
CREATE TABLE paper_drafts (
    id SERIAL PRIMARY KEY,
    title VARCHAR(500),
    abstract TEXT,
    content JSONB,  -- {sections: [{title, content}]}
    outline JSONB,
    references JSONB,
    status VARCHAR(20),  -- draft/reviewing/published
    project_id INTEGER REFERENCES projects(id),
    author_id INTEGER REFERENCES members(id),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- 写作建议表
CREATE TABLE writing_suggestions (
    id SERIAL PRIMARY KEY,
    draft_id INTEGER REFERENCES paper_drafts(id),
    suggestion_type VARCHAR(50),  -- grammar/style/citation/format
    original_text TEXT,
    suggested_text TEXT,
    position JSONB,  -- {start, end}
    status VARCHAR(20),  -- pending/accepted/rejected
    created_at TIMESTAMP DEFAULT NOW()
);

-- 基金申请表
CREATE TABLE grant_applications (
    id SERIAL PRIMARY KEY,
    title VARCHAR(500),
    funding_agency VARCHAR(200),
    content JSONB,
    budget JSONB,
    status VARCHAR(20),
    deadline DATE,
    applicant_id INTEGER REFERENCES members(id),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);
```
