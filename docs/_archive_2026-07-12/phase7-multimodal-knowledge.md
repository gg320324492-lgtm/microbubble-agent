## 7.1 图片理解与提取

### 7.1.1 PDF 图片提取增强
**当前状态**: `_upload_knowledge_images` 已实现基础图片上传
**目标**: 完善图片提取、存储、关联链路

| 任务 | 技术方案 | 工时 |
|------|----------|------|
| PDF 图片提取优化 | PyMuPDF `page.get_images()` 提取嵌入图片 | 2d |
| 图片格式转换 | 统一转为 PNG，压缩到 1MB 以内 | 1d |
| MinIO 存储 | `knowledge/{id}/fig_{idx}.png` 前缀 | 1d |
| 图片 URL 嵌入 | 替换 `[FIGURE:x]` 占位符为 Markdown 图片语法 | 1d |
| 图片懒加载前端 | `loading="lazy"` + 骨架屏 | 1d |

**验收标准**: 上传 PDF 后，知识详情页正确显示所有嵌入图片

### 7.1.3 图片内容描述生成

## 技术风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| OCR 准确率低 | 图片文字提取不准 | PaddleOCR + 人工校正 |
| 公式识别错误 | 公式库质量下降 | 置信度阈值 + 人工审核 |
| CLIP 模型太大 | 服务器资源不足 | 使用轻量版 CLIP-ViT-B/32 |
| 多模态搜索慢 | 用户体验差 | 索引优化 + 缓存 |
